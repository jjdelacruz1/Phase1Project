About Us

Learn more about the team who created this website, how and why this project was started, and how to help out!

We are a team of 3 students attending the DigitalCrafts Flex Program. Our team consists of Jerome Dela Cruz, Alvin Ng, and Sue Park.

The purpose of this website is to help people find GREAT happy hours in their area by searching with their zip code or their city.

The technologies that we are using to build this website are the following:

  • HTML, CSS, & JavaScript
  • Bootstrap 4
  • Yelp and MapBox APIs
  • Bideo.js
  • Surge for Deployment

We also wanted to create a template that would allow others to use this code and make ANY kind of finder in their area! All you would need to do is change out "happy hour" for something like "sushi" or "pizza" and then your site would be good to go!
